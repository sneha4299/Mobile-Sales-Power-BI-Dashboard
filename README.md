# Mobile-Sales-Power-BI-Dashboard
Power BI dashboard analyzing mobile sales performance across cities, brands, payment methods, and time periods.

Overview
This project presents an interactive Power BI dashboard to analyze mobile sales data across multiple dimensions such as city, brand, payment method, and time period. The dashboard provides key performance indicators (KPIs), comparative charts, and trend analysis to support business decision-making.

Objectives
- Track total sales, transactions, and average price
- Compare performance by city, brand, and mobile model
- Analyze payment method preferences
- Monitor monthly and quarterly sales trends
- Evaluate customer ratings and satisfaction

Dataset
- Key fields: City, Brand, Mobile Model, Payment Method, Quantity, Sales Amount, Rating

Tools & Skills Used
- Power BI: Data modeling, DAX measures, interactive dashboards
- Visualization: Maps, bar charts, line charts, pie charts
- Business analysis: Sales performance, customer behavior insights

Key Insights
- Total sales reached 769M with 19K units sold
- UPI, debit card, credit card, and cash payments are evenly distributed (~25% each)
- Apple, Samsung, and OnePlus are top-performing brands
- Sales peak in certain months and vary across cities like Delhi, Mumbai, and Bangalore
- Customer ratings show majority “Good” feedback, with some “Average” and “Poor”

How to Use
- Download the `.pbix` file from this repository
- Open in Power BI Desktop
- Explore filters and interactive visuals to analyze mobile sales trends

Conclusion
This dashboard demonstrates how Power BI can be used to analyze retail sales data, uncover customer behavior patterns, and provide actionable insights for business growth.
